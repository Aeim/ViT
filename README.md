# Vision Transformer in Pytorch

Learn/Refresh about Vision Transformer architecture
![ViT Model](ViT_model.png)
## Requirement
- pytorch
- matplotlib
- einops

## Step to run
```
# create environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
Then run cells in Jupyter notebook
